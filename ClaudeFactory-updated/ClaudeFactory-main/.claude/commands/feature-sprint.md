# Feature Sprint

---

## Command

```
---
name: feature-sprint
description: End-to-end feature development sprint with TDD
---

Run the Feature Sprint workflow defined in `.claude/skills/workflows/feature-sprint-workflow.md`.

Follow every phase in order, respecting gates and agent personas. Use the full workflow instructions — do not skip or summarize phases.

Input (feature description or path to a file containing it):
$ARGUMENTS

```

## Arguments

- `{ARGUMENTS}`: feature description or path to a file containing it
