# Tdd Cycle

---

## Command

```
---
name: tdd-cycle
description: Run a strict Red-Green-Refactor TDD cycle
---

Run the TDD Cycle workflow defined in `.claude/skills/workflows/tdd-cycle-workflow.md`.

Follow every phase in order, respecting gates and agent personas. Use the full workflow instructions — do not skip or summarize phases.

Input (feature or bug description, or path to a file containing it):
$ARGUMENTS

```

## Arguments

- `{ARGUMENTS}`: feature or bug description, or path to a file containing it
