# Design Sprint

---

## Command

```
---
name: design-sprint
description: Run a design sprint workflow to rapidly ideate, prototype, and validate a feature concept
---

Run the Design Sprint workflow defined in `.claude/skills/workflows/design-sprint-workflow.md`.

Follow every phase in order, respecting gates and agent personas. Use the full workflow instructions — do not skip or summarize phases.

Input (the feature or problem to design-sprint on):
$ARGUMENTS

```

## Arguments

- `{ARGUMENTS}`: the feature or problem to design-sprint on
