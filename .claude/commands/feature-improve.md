# Feature Improve

---

## Command

```
---
name: feature-improve
description: Improve an existing feature with safe, tested changes
---

Run the Feature Improve workflow defined in `.claude/skills/workflows/feature-improve-workflow.md`.

Follow every phase in order, respecting gates and agent personas. Use the full workflow instructions — do not skip or summarize phases.

Input (feature name + description of the improvement to make):
$ARGUMENTS

```

## Arguments

- `{ARGUMENTS}`: feature name + description of the improvement to make
