# Feature Spec

---

## Command

```
---
name: feature-spec
description: Turn a raw idea into a complete feature specification
---

Run the Feature Spec workflow defined in `.claude/skills/workflows/feature-spec-workflow.md`.

Follow every phase in order, respecting gates and agent personas. Use the full workflow instructions — do not skip or summarize phases.

Input (feature idea or description to spec out):
$ARGUMENTS

```

## Arguments

- `{ARGUMENTS}`: feature idea or description to spec out
