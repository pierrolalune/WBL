# Idea To Feature

---

## Command

```
---
name: idea-to-feature
description: Turn a feature idea into a fully built, tested, shipped feature
---

Run the Idea-to-Feature workflow defined in `.claude/skills/workflows/idea-to-feature-workflow.md`.

Follow every phase in order, respecting gates, HITL checkpoints, and agent personas. Use the full workflow instructions — do not skip or summarize phases.

Input (feature idea or path to a file containing it):
$ARGUMENTS

```

## Arguments

- `{ARGUMENTS}`: feature idea or path to a file containing it
